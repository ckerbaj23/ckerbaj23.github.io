# ckerbaj23.github.io

Personal portfolio and blog for Charles Kerbaj — finance student at Bryant University with a focus on markets, AI, and automation.

## What's here

- **Homepage** (`index.html`) — hero section, project portfolio, animated financial ticker splash screen, dark mode
- **Blog posts** — build logs and write-ups for personal projects:
  - `post-printer.html` — Daily Report Printer
  - `post-trading.html` — Trading systems and strategy
  - `post-youtube.html` — YouTube automation project
- **Resume** — `Charles_Kerbaj_Resume.pdf`

## Stack

Vanilla HTML, CSS, and JavaScript. No frameworks, no build step — just static files served via GitHub Pages.

**Fonts:** Cormorant Garamond (serif) + Jost (sans-serif) via Google Fonts

## Running locally

Open `index.html` in a browser directly, or use any static file server:

```bash
npx serve .
```
